# 바늘구멍(핀홀) 상 시뮬레이터

광원 → 구멍 → 스크린 장치에서 **거리·구멍 크기·광원 모양·구멍 모양**을 바꿔가며,
스크린에 맺히는 상의 **뒤집힘 · 배율 · 번짐**을 실시간으로 관찰하는 중학생용 교육 시뮬레이터입니다.

빌드 과정 없이 순수 **HTML / CSS / JavaScript + `<canvas>`** 로만 작성되어 GitHub Pages에서 바로 동작합니다. (외부 라이브러리 없음)

## 사용법
- `index.html` 을 브라우저로 열면 됩니다.
- GitHub Pages: 저장소 **Settings → Pages → Branch** 를 이 브랜치(또는 main)로 지정하면 공개됩니다.

## 물리 모델
- 좌표: 구멍 평면이 원점, 광원은 뒤쪽 거리 `d_o`, 스크린은 앞쪽 거리 `d_i`.
- 배율 **m = d_i / d_o** (상은 광원을 m배 하고 상하·좌우 반전)
- 번짐 배율 **s = 1 + d_i / d_o** (구멍 모양을 s배 한 크기로 각 점이 번짐)
- 스크린 상 = **(m배·반전한 광원 모양) ⊛ (s배 확대한 구멍 모양)** 의 합성곱(convolution)

합성곱은 둘 중 큰 쪽(광원 상 또는 번짐 커널)을 연속 스프라이트로,
작은 쪽을 격자 샘플로 만들어 각 샘플 위치에 가산 합성하고 밝기를 자동 정규화하는
방식으로 저해상도(220×220)에서 계산합니다.

## 검증되는 핵심 현상 (조절판 하단 프리셋)
1. `d_o = d_i` → 배율 1배, 상은 180° 뒤집힘
2. 구멍이 커질수록 번짐(흐림)이 커져 상이 뭉개짐
3. 광원을 아주 멀리 → 상이 작아지고 광원 대신 **구멍 모양**(예: 별)이 찍힘
4. 구멍 모양을 바꾸면 번짐(흐릿한 윤곽)이 그 모양을 따름

## 파일
- `index.html` — 화면 구성 / 컨트롤
- `style.css` — 반응형 스타일(모바일·PC)
- `script.js` — 물리 모델 · 합성곱 렌더링 · 모식도
